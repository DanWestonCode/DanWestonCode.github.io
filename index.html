    Volume Renderer

    Source Code can be found on my GitHub here

    About:

    Volume Renderer implemented with C++/DirectX 11. This renders scalar data without the use of fitting proper geometry to the data. The technique works by rendering a cube. The front and back faces of the cube are rendered to separate Render Textures. A Pixel shader is then used to iterate through the scalar data, casting a ray from the front render texture to the back. The scalar data is sampled at equal intervals along the way. Each sample accumulates a result which is rendered onto the cube.

    Technologies:

    C++, HLSL, DirectX 11

    Flow Maps

    Source Code can be found on my GitHub here

    About:

    This tech demo illustrates the use of Flow Maps within a shader to mimic directional water flow. The demo is written in TypeScript and rendered in WebGL using Pixi. This technique uses a hand painted normal map to depict directional vectors. Normals are unrolled and integrated with a timestep to move them in their painted direction. The outcome of the integrated normal is used to sample the water texture and give the illusion of flow.

    As well as water flow, this technique can be used for multiple effects including lava, weather patterns and traffic!

    I mainly wrote this demo to familiarise myself writing, loading and working with shaders using Pixi and WebGL. The implementation is based off of Kyle Haywards article.

    Technologies:

    TypeScript, WebGL, GLSL
